
# 🦋 Butterfly Pairing

Site de génération de session ID pour le bot WhatsApp **BUTTERFLY 16 MD**.

## Fonctionnalités

- Génération de code de jumelage (pairing code) à 8 chiffres.
- Récupération automatique de la `SESSION_ID` au format `butterfly~...`.
- Interface web simple et intuitive.
- Déploiement facile sur Render.
